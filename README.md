# gh-graduation-project

## Installation

```bash
$ pm run api:install-dependencies
or
$ yarn

$ npm run client:install-dependencies
```

## Running the database

```bash
$ docker-compose up database
```

## Running the app

```bash
$ npm run client:build

$ npm run api:start
or
$ yarn api:start

```
